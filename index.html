Built as identity infrastructure.

A modular architecture for registering DIDs, issuing credentials, holding identity assets, delegating authority, and verifying trust across systems.

1

DID Registry

Create and resolve identifiers for people, agents, APIs, skills, devices, and organizations.

2

VC Issuer

Issue verifiable credentials that bind claims, authority, and trust to identities.

3

ID Wallet

Store credentials and identity material for users, operators, and participating entities.

4

Delegation Layer

Grant scoped permissions to agents, APIs, and skills with revocation and auditability.

Developer quick start.

Run the platform locally and explore the DID Registry, VC Issuer, and Wallet services.

git clone https://github.com/didoneworld/platform.git
cd platform
docker-compose up

Local services

DID Registry: http://localhost:8000

VC Issuer: http://localhost:8001

Wallet: http://localhost:3000
